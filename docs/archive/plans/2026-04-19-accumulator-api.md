# Accumulator API Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement a local-only accumulator primitive in `incr/` that lets tracked memo computations push values into a side channel with correct incremental invalidation — the driver is lambda type-checker diagnostics.

**Architecture:** Handle-local typed storage (`Accumulator[T]` owns `per_memo : HashMap[CellId, Array[T]]`), runtime stores `Array[SlotMeta]` with type-erased closures captured at construction time with `T : Eq` in scope. Invalidation piggybacks on the ordinary `pull_verify` walk via per-memo `push_revised_at : Revision` bumps inside a unified `finalize_memo` closure. Memo compute closures widen from `() -> T` to `() -> T raise Failure` (non-raising closures auto-promote, empirically verified).

**Tech Stack:** MoonBit (wasm-gc target), `moonbitlang/core/hashmap`, `moonbitlang/core/hashset`. Tests via `moon test`; `moon check`, `moon info`, `moon fmt` for CI checks.

**Status:** Shipped 2026-04-20. See [ADR](../../decisions/2026-04-20-accumulator-api.md).

**Spec:** `docs/archive/completed-phases/2026-04-19-accumulator-api-design.md`

```moonbit
///|
/// Internal: guarantee a memo has been computed and is up-to-date in the
/// current revision, WITHOUT recording an ordinary dep on the caller's
/// tracking frame. Used by Memo::accumulated.
///
/// # Behavior
///
/// - If the cell is disposed, returns Ok(()) silently (caller handles
///   disposal separately; this is defense-in-depth for other call sites).
/// - If the memo has never been computed (`value is None`), triggers the
///   first compute via the memo's stored compute closure.
/// - Otherwise runs `pull_verify`, which may recompute if stale.
/// - Neither path calls `record_dependency`. The synthetic accumulator
///   dep is recorded separately in the accumulator read flow.
///
/// # Returns
///
/// `Ok(())` on success (including disposed — silently), `Err(CycleError)`
/// if the memo is in a cycle.
fn Runtime::ensure_computed_untracked(
  self : Runtime,
  cell_id : @incr_types.CellId,
) -> Result[Unit, @incr_types.CycleError] {
  if self.is_cell_disposed(cell_id) {
    return Ok(())
  }
  let data = self.get_memo_data(cell_id)

  // meta.changed_at as side effects. It does NOT call record_dependency.

  // fresh runtime starts at initial revision, so we cannot short-circuit
  // via pull_verify's fast path at that point (it would incorrectly mark
  // the memo verified without ever calling compute — see verify.mbt:92).
  if data.verified_at == @incr_types.Revision::initial() {
    match data.compute() {
      Ok(_) => Ok(())
      Err(e) => Err(e)
    }
  } else {
    match self.pull_verify(cell_id) {
      Ok(()) => Ok(())
      Err(e) => Err(e)
    }
  }
}
```

```moonbit
/// A side-channel collector for values pushed by memo compute closures.
///
/// Typed buffers (`per_memo`, `prev_push_sets`) live on the handle; runtime
/// holds only type-erased closures in SlotMeta. See the design doc
/// `docs/archive/completed-phases/2026-04-19-accumulator-api-design.md`.
pub struct Accumulator[T] {
  priv rt : Runtime
  priv slot_id : @incr_types.AccumulatorId
  priv label : String?
  // Typed per-memo buffer: the values this memo pushed in its latest
  // successful compute. Read by Memo::accumulated(_peek).
  priv per_memo : @hashmap.HashMap[@incr_types.CellId, Array[T]]
  // Snapshot of per_memo taken at BEFORE_CLOSURE; restored on ON_ABORT.
  priv prev_push_sets : @hashmap.HashMap[@incr_types.CellId, Array[T]]
  // Per-memo revision bumped in finalize_memo when push-set differs
  // across recomputes. Reader records this in its accumulator_reads.
  priv push_revised_at : @hashmap.HashMap[
    @incr_types.CellId,
    @incr_types.Revision,
  ]

  fn[T : Eq] new(rt~ : Runtime, label? : String) -> Accumulator[T]
}

  label? : String,

  let per_memo : @hashmap.HashMap[@incr_types.CellId, Array[T]] = @hashmap.new()
  let prev_push_sets : @hashmap.HashMap[@incr_types.CellId, Array[T]] = @hashmap.new()
  let push_revised_at : @hashmap.HashMap[
    @incr_types.CellId,
    @incr_types.Revision,
  ] = @hashmap.new()
  // The SlotMeta closures capture these typed fields (T : Eq in scope here).
  // They'll be wired in Task 11; for now, stub with no-ops so registration
  // succeeds.

    label,
    disposed: false,
    snapshot_and_clear: _ => (),
    restore_buffer: _ => (),

    dispose_memo: _ => (),

}

///|
/// Returns the optional label provided at construction.
pub fn[T] Accumulator::label(self : Accumulator[T]) -> String? {
  self.label
}

///|
/// Returns true iff this accumulator has been disposed.
pub fn[T] Accumulator::is_disposed(self : Accumulator[T]) -> Bool {
  self.rt.accumulator_slots[self.slot_id.id].disposed
}

///|
/// Disposes the accumulator. Idempotent. Clears typed buffers and removes
/// reverse-index entries from rt.accumulator_contributions.
pub fn[T] Accumulator::dispose(self : Accumulator[T]) -> Unit {
  let slot = self.rt.accumulator_slots[self.slot_id.id]
  if slot.disposed {
    return
  }
  // Remove slot_id from every contributing memo's reverse index.
  for m_id in self.per_memo.keys() {
    match self.rt.accumulator_contributions.get(m_id) {
      Some(set) => {
        set.remove(self.slot_id)
        if set.is_empty() {
          self.rt.accumulator_contributions.remove(m_id)
        }
      }
      None => ()
    }
  }
  self.per_memo.clear()
  self.prev_push_sets.clear()
  self.push_revised_at.clear()
  slot.disposed = true
  // slot_id stays allocated — monotonic, never reused.
}

/// Debug representation.

  let lbl = match self.label {
    Some(s) => "\"" + s + "\""
    None => "None"
  }
  "Accumulator(id=" +
  self.slot_id.id.to_string() +
  ", label=" +
  lbl +
  ", disposed=" +
  self.is_disposed().to_string() +
  ")"
}
```

```moonbit
///|
/// Appends a value to this accumulator, keyed by the currently-computing
/// memo's CellId.
///
/// # Errors
///
/// - `fail` if called outside a tracked compute context
/// - `fail` if the top tracking frame is not a Memo (MVP restriction)
/// - `fail` if this accumulator has been disposed
/// - `abort` (via `check_cross_runtime`) if slot belongs to a different runtime
pub fn[T] Accumulator::push(self : Accumulator[T], value : T) -> Unit raise Failure {
  // Cross-runtime check (abort — pre-existing pattern).
  self.rt.check_cross_runtime("Accumulator")
  // Disposed check.
  let slot = self.rt.accumulator_slots[self.slot_id.id]
  if slot.disposed {
    fail("push to disposed Accumulator")
  }
  // Top-frame check.
  let frame = match self.rt.top_active_query() {
    Some(q) => q
    None => fail("Accumulator::push called outside a tracked compute")
  }
  let m_id = frame.cell_id
  // Top-frame must be a PullMemo / HybridMemo cell.
  match self.rt.core.cell_index[m_id.id] {
    PullMemo(_) | HybridMemo(_) => ()
    _ => fail("Accumulator::push only valid inside Memo compute")
  }
  // Append to typed buffer.
  let buf = match self.per_memo.get(m_id) {
    Some(existing) => existing
    None => {
      let fresh : Array[T] = []
      self.per_memo.set(m_id, fresh)
      fresh
    }
  }
  buf.push(value)
  // Stage touched slot on ActiveQuery (committed in AFTER_CLOSURE).
  frame.touched_accumulator_slots.add(self.slot_id)
}
```

```moonbit
pub fn[T : Eq] Accumulator::new(
  rt~ : Runtime,
  label? : String,
) -> Accumulator[T] {
  let per_memo : @hashmap.HashMap[@incr_types.CellId, Array[T]] = @hashmap.new()
  let prev_push_sets : @hashmap.HashMap[@incr_types.CellId, Array[T]] = @hashmap.new()
  let push_revised_at : @hashmap.HashMap[
    @incr_types.CellId,
    @incr_types.Revision,
  ] = @hashmap.new()

  // snapshot_and_clear: move current buffer into prev, replace per_memo[M]
  // with a fresh array. Used in BEFORE_CLOSURE.
  let snapshot_and_clear = (m_id : @incr_types.CellId) => {
    match per_memo.get(m_id) {
      Some(current) => {
        prev_push_sets.set(m_id, current)
        per_memo.set(m_id, [])
      }
      None => {
        // Memo had no prior pushes; record empty prev so diff catches
        // first-ever pushes correctly.
        prev_push_sets.set(m_id, [])
      }
    }
  }

  // restore_buffer: undo snapshot. Used in ON_ABORT.
  let restore_buffer = (m_id : @incr_types.CellId) => {
    match prev_push_sets.get(m_id) {
      Some(prev) => {
        per_memo.set(m_id, prev)
        prev_push_sets.remove(m_id)
      }
      None => {
        // Shouldn't happen — BEFORE_CLOSURE always writes prev_push_sets.
        ()
      }
    }
  }

  // finalize_memo: atomic commit-phase work. Compare prev vs current;
  // bump push_revised_at if different; drop snapshot; gc empty buffer.
  let finalize_memo = (m_id : @incr_types.CellId, rev : @incr_types.Revision) => {
    let prev = match prev_push_sets.get(m_id) {
      Some(p) => p
      None => []
    }
    let current = match per_memo.get(m_id) {
      Some(c) => c
      None => []
    }
    // Structural Array[T] equality (T : Eq in scope from outer fn).
    let same = arrays_eq(prev, current)
    if !same {
      push_revised_at.set(m_id, rev)
    }
    prev_push_sets.remove(m_id)
    if current.length() == 0 {
      per_memo.remove(m_id)
    }
  }

  // dispose_memo: remove memo's entries when the memo itself is disposed.
  let dispose_memo = (m_id : @incr_types.CellId) => {
    per_memo.remove(m_id)
    prev_push_sets.remove(m_id)
    push_revised_at.remove(m_id)
  }

  let push_revised_at_for = (m_id : @incr_types.CellId) -> @incr_types.Revision {
    match push_revised_at.get(m_id) {
      Some(r) => r
      None => @incr_types.Revision::initial()
    }
  }

  let meta : SlotMeta = {
    label,
    disposed: false,
    snapshot_and_clear,
    restore_buffer,
    finalize_memo,
    dispose_memo,
    push_revised_at_for,
  }
  let slot_id = rt.register_accumulator_slot(meta)
  { rt, slot_id, label, per_memo, prev_push_sets, push_revised_at }
}

///|
/// Structural equality over Array[T] where T : Eq.
fn[T : Eq] arrays_eq(a : Array[T], b : Array[T]) -> Bool {
  if a.length() != b.length() {
    return false
  }
  for i in 0..<a.length() {
    if a[i] != b[i] {
      return false
    }
  }
  true
}
```

```moonbit
fn[T] Runtime::memo_force_recompute(
  self : Runtime,
  cell_id : CellId,
  compute_fn : () -> T raise Failure,
  old_value : T?,
  eq_fn : (T, T) -> Bool,
) -> Result[T, CycleError] raise Failure {
  let cell = self.get_memo_data(cell_id)
  if cell.in_progress {
    return Err(
      CycleError::from_path(
        self,
        self.collect_tracking_path(),
        cell.meta.cell_id,
      ),
    )
  }
  cell.in_progress = true

  // BEFORE_CLOSURE: snapshot each accumulator slot that M contributed to
  // in its previous successful compute. Clone the set — prev_contributions
  // is iterated after compute and must not reflect touched_accumulator_slots.
  let prev_contributions : Array[@incr_types.AccumulatorId] =
    match self.accumulator_contributions.get(cell_id) {
      Some(s) => s.iter().to_array()
      None => []
    }
  for slot_id in prev_contributions {
    let slot = self.accumulator_slots[slot_id.id]
    if !slot.disposed {
      (slot.snapshot_and_clear)(cell_id)
    }
  }

  let old_deps = cell.dependencies
  self.push_tracking(cell_id)

  // Run user compute. On raise → ON_ABORT path.
  let new_value = try compute_fn() catch {
    e => {
      // Extract the still-live top frame to see which slots got touched
      // during this aborted run.
      let frame = self.top_active_query()
      let touched : Array[@incr_types.AccumulatorId] = match frame {
        Some(q) => q.touched_accumulator_slots.iter().to_array()
        None => []
      }
      // ON_ABORT.1: restore snapshots for every slot in prev_contributions.
      for slot_id in prev_contributions {
        let slot = self.accumulator_slots[slot_id.id]
        if !slot.disposed {
          (slot.restore_buffer)(cell_id)
        }
      }
      // ON_ABORT.2: for slots touched during this run but not in prev,
      // clear the new-run-only buffer to avoid leaking partial pushes.
      for slot_id in touched {
        if prev_contributions.contains(slot_id) {
          continue
        }
        let slot = self.accumulator_slots[slot_id.id]
        if !slot.disposed {
          (slot.dispose_memo)(cell_id)
        }
      }
      // Pop frame (discards staged deps + accumulator_reads).
      let _ = self.pop_tracking_full()
      cell.in_progress = false
      raise e
    }
  }

  // Normal path — pop the full frame to access accumulator state.
  let query = self.pop_tracking_full()
  let new_deps = query.dependencies
  let new_seen = query.seen
  let touched = query.touched_accumulator_slots
  let staged_reads = query.accumulator_reads

  // AFTER_CLOSURE: finalize each slot in prev ∪ touched (atomic diff +
  // bump + cleanup per slot), and rebuild accumulator_contributions from
  // slots that still have a buffer.
  let all_slots : @hashset.HashSet[@incr_types.AccumulatorId] = @hashset.new()
  for s in prev_contributions {
    all_slots.add(s)
  }
  for s in touched.iter() {
    all_slots.add(s)
  }
  let current_revision = self.core.revision.current_revision
  let new_contributions : @hashset.HashSet[@incr_types.AccumulatorId] = @hashset.new()
  for slot_id in all_slots.iter() {
    let slot = self.accumulator_slots[slot_id.id]
    if slot.disposed {
      continue
    }
    (slot.finalize_memo)(cell_id, current_revision)
    if (slot.has_buffer_for)(cell_id) {
      new_contributions.add(slot_id)
    }
  }

  // Commit staged_reads into MemoData.accumulator_reads (overwrite any
  // prior-run entries; a successful recompute is the authoritative state).
  cell.accumulator_reads.clear()
  for entry in staged_reads.iter() {
    cell.accumulator_reads.set(entry.0, entry.1)
  }
  // Commit rebuilt contributions.
  if new_contributions.is_empty() {
    self.accumulator_contributions.remove(cell_id)
  } else {
    self.accumulator_contributions.set(cell_id, new_contributions)
  }

  // Existing epilogue — subscribers, durability, backdate, verified_at.
  let deps_changed = self.diff_and_update_subscribers(
    cell_id,
    old_deps,
    new_deps,
    new_seen~,
  )
  cell.dependencies = new_deps
  if deps_changed {
    cell.meta.durability = compute_durability(self, new_deps)
  }
  let value_changed = match old_value {
    None => true
    Some(old) => !eq_fn(old, new_value)
  }
  if value_changed {
    cell.meta.changed_at = self.core.revision.current_revision
  }
  cell.verified_at = self.core.revision.current_revision
  cell.in_progress = false
  Ok(new_value)
}
```

```moonbit
///|
/// Tracked read: returns the accumulated values pushed by `self` during
/// its last successful compute, records a synthetic dep on the current
/// compute frame, and forces verification of `self` via
/// `ensure_computed_untracked`.
///
/// # Errors
///
/// - `fail` if accumulator is disposed
/// - `fail` if target memo (self) is disposed
/// - raises `CycleError` if `self` is in a cycle
pub fn[T, A] Memo::accumulated(
  self : Memo[T],
  acc : Accumulator[A],
) -> Array[A] raise {
  self.rt.check_cross_runtime("Memo::accumulated")
  let slot = self.rt.accumulator_slots[acc.slot_id.id]
  if slot.disposed {
    fail("Memo::accumulated called on disposed Accumulator")
  }
  if self.rt.is_cell_disposed(self.cell_id) {
    fail("Memo::accumulated called on disposed target memo")
  }
  // Force verification of target memo without recording an ordinary dep.
  match self.rt.ensure_computed_untracked(self.cell_id) {
    Ok(()) => ()
    Err(e) => raise e
  }
  // Stage synthetic read on current frame (if any).
  let current_rev = (slot.push_revised_at_for)(self.cell_id)
  match self.rt.top_active_query() {
    Some(frame) =>
      frame.accumulator_reads.set((acc.slot_id, self.cell_id), current_rev)
    None => ()
  }
  // Return defensive copy of buffer.
  match acc.per_memo.get(self.cell_id) {
    Some(vs) => vs.copy()
    None => []
  }
}
```

```moonbit
///|
/// Creates an accumulator owned by this scope. The accumulator is disposed
/// automatically when the scope is disposed (via `dispose_hooks`).
pub fn[T : Eq] Scope::accumulator(
  self : Scope,
  label? : String,
) -> Accumulator[T] {
  guard !self.disposed else {
    abort("Scope::accumulator called on a disposed scope")
  }
  let a = Accumulator::new(rt~=self.runtime, label?)
  self.dispose_hooks.push(() => a.dispose())
  a
}
```

```moonbit
  let rt = Runtime::new()
  let should_fail = Signal::new(rt, false)
  let acc_a : Accumulator[Int] = Accumulator::new(rt~=rt)

    _ => ()

  let data_after = rt.get_memo_data(consumer.id())
```
